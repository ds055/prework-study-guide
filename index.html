HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user.
  • The meta elements, found in the parent head element, contains info for the browser.
  • Your page should only have 1 "h1" level heading.
  • Semantic elements are easier to read for others looking at your code.
