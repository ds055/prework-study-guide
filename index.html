Prework Study Guide

HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user.
  • The meta elements, found in the parent head element, contains info for the browser.
  • Your page should only have 1 "h1" level heading.
  • Semantic elements are easier to read for others looking at your code.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Use an external CSS style sheet to help use DRY Code.
  • Separation of concerns: separate coding into files based on use
  • The asterisk wild card symbol can be used to apply CSS to all elements.

Git

  • git status: checks out what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • Add notes to git commits to show what changes were made for this commit.
  • Changes must be staged before they can be committed.
  • git pull origin main: makes the local main branch match the main branch online
